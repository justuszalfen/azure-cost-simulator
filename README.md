# ☁️ Azure Cost Simulator - Systemhaus Edition

Willkommen beim ultimativen Stresstest für angehende Cloud-Administratoren!

Das Spiel:
Du bist der neue Admin. Der Kunde will maximale Skalierung für seinen Online-Shop, dein Chef will knallharte Effizienz. 
Plötzlich trifft ein viraler Hit deinen Server und der Traffic geht durch die Decke! Hast du das Autoscale richtig konfiguriert? 
Hast du an das rettende Kosten-Limit gedacht?
Finde es heraus, bevor das Monatsbudget in Sekundenbruchteilen verbrannt ist und du (virtuell) gefeuert wirst.

Warum dieses Projekt?
Dieses Mini-Game ist als praktisches Begleitprojekt zu meiner AZ-900 Vorbereitung entstanden. 
Es überführt die Theorie der Azure-Kostenfaktoren (Ressourcentypen, Regionen, Autoscale-Limits) 
in eine interaktive, gemeine kleine Simulation. 

Features:
- 💸 Realistische Budget-Verbrennung in Echtzeit
- 📈 Fiese CPU-Spikes und versteckte Autoscale-Fallen
- 😭 Game-Over-Screens direkt aus der Hölle der IT-Buchhaltung

*Let's deploy!*
